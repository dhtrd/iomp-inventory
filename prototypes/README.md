# نماذج التصميم (Design Prototypes) — فهرس

## نظام إدارة الجرد متعدد المستخدمين (IOMP) — شركة الضبيبي التجارية

هذا الفهرس يوثّق نماذج التصميم الحيّة (HTML مستقلّة) التي صُمِّمت وتُحقِّق منها آليًّا لكل مرحلة من مراحل م١١–م٢٦، والتي على أساسها نُفِّذ **الربط الفعلي** في `index.html` (الدفعات ر١–ر١٠ + مركز الإدارة م١٨). النماذج الكاملة محفوظة في مشروع claude.ai المرافق (وثائق `claude/نموذج-*.html`)، وتُستحضر عند الحاجة للمرجعية البصرية أو لإعادة بناء أي منظومة.

|---|---|---|---|

## ملاحظة معمارية

كل منظومة رُبطت **خلف علمٍ افتراضه معطّل** (حيث تغيّر مسارًا قائمًا)، وبأنماط التطبيق نفسها (RTL/داكن/جوال) وبصفر كلفة (بلا مكتبات أو خدمات جديدة). التفاصيل الكاملة في `../docs/دليل-المعمارية-والربط.md`. النماذج هنا مرجعية تصميميّة؛ **المصدر النهائي المنفَّذ هو `../index.html`** واختباراته (`../test_*.mjs`، ٣٩٩ اختبارًا).
